Utils Module
===========

.. module:: pita_algorithm.utils
   :synopsis: Utility functions supporting simulation setup and execution in the PITA Algorithm.

The `Utils` class encompasses a collection of static utility functions designed to facilitate various tasks within the PITA Algorithm framework. These functions include, but are not limited to, parameter randomization and coordinate adjustments relative to defined boundaries.

Usage
-----

The `Utils` class provides static methods that can be directly invoked.

.. autoclass:: pita_algorithm.utils.Utils
   :members:
   :undoc-members:
   :show-inheritance:

Key Functionalities
-------------------

- **Randomization Parameter Retrieval**: Extracts randomization parameters from a configuration dictionary, supporting diverse simulation dynamics.

- **Coordinate Offset and Scaling**: Adjusts object coordinates to ensure they fit within the designated area boundaries, crucial for accurate object placement.
