PITA Algorithm Base Package
==========================

.. module:: pitapy.base
   :synopsis: Overview of the base module of the PITA Algorithm framework.

The `pitapy.base` package provides the foundational classes and functions utilized across the PITA Algorithm framework. This module includes core functionality for object modeling, simulation environment setup, and base components for asset placement and rule evaluation.
