ConfigReader Module
==================

.. module:: pita_algorithm.utils
   :synopsis: Utility class for reading YAML configuration files.

The `ConfigReader` class is a utility within the PITA Algorithm framework designed to read YAML files containing user-defined configurations. It parses the YAML file and returns a dictionary representing the configuration settings, facilitating the initialization of simulations based on these user specifications.

Usage
-----

The `ConfigReader` class offers a straightforward static method `execute` to read a YAML configuration file and return its contents as a dictionary. This method requires the path to the YAML file as its argument.
