Asset Placement Rules Package
===================================================

The `pitapy.base.asset_placement.rules` package encompasses a collection of rule classes that define constraints for placing objects within the simulation environments. These rules are pivotal for ensuring that object placements adhere to specific requirements or constraints, enhancing the realism and effectiveness of simulations.
