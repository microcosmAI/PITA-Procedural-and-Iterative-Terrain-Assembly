World Sites Module
=======================================

.. module:: pitapy.base.world_sites
   :synopsis: Overview of the world sites module within the PITA Algorithm framework.

The `pitapy.base.world_sites` module encompasses the classes and functionalities that define and manage different simulation sites within the PITA Algorithm framework. This includes abstract definitions of sites, specific implementations for environments and areas, and utilities for managing these sites within simulations.
